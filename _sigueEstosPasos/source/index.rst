Tutorial de Sphinx con Python
=============================

Sphinx fue generado para crear documentación de Python.
Genera documentación HTML, LaTeX (para versiones PDF), ePub, Texinfo, manual pages, plain text.
Sphinx usa reStructuredText como lenguaje de marcas.
Ayuda mucho a la documentación, pero no todo se genera de una forma totalmente automática
por lo que tendrás que modificar algunos ficheros 'a mano'.
