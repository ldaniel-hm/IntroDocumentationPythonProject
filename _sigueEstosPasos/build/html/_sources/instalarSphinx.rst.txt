:index:`Instalar Sphinx`
=========================

Debe instalar Sphinx en su entorno virtual. Hay dos formas de hacerlo:

* A través de la Configuración de PyCharm
    1. PyCharm > Preferences > Project:documentationPythonProject > Python Interpreter
    2. Pulsar ``+``
    3. Buscar ``Sphinx``
    4. Seleccionar ``Install Package``

* A través del Terminal de PyCharm
    1. View > Tool Windows > Terminal
    2. Asegurate que aparece (venv) en el terminal
    3. Escribe ``pip3 install sphinx``

* Usa los mismos pasos para instalar otros paquetes, si fueran necesarios.
    P.e. instalar el paquete de estilo ``sphinx_rtd_theme``.
